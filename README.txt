Scala Simple Websocket Based Chatroom
This is a simple Websocket based chatroom, it purpose is to demonstrate scala and it functionality through a real life demo that people can see the inner workings of scala

Functionality
You should be able to start the server and open a http://localhost:9000 on your favorite search browser
You should then be able to login and it will redirect you to a chat form, note that if you change the ID of the chatroom in the url when you submit your messages nothing happens, this is due to the JWT verification on the server
You should be able to open this on multiple incognito windows of your browser and as long as the chat ID is the same, you should be able to see the messages populate on your screen

Starting the Server
Install sbt
run the sbt command
run the run command in the sbt terminal
